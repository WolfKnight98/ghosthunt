Welcome to GitHub Pages.

GhostHunt is a gamemode for Garry's Mod that aims to improve the level of immersion when playing on ghost hunt/horror maps, it does this by having special features and offering server owners console commands to control what players can do. It is currently free and will always remain free, anyone can edit and re-upload the gamemode but make sure you credit me!

Authors and Contributors
